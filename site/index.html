Discord Bot

A multi-purpose bot for Discord, coded in discord.js

COMMANDS

General Commands

Commands that don't really fit into a category

ping: Shows the latency and API latency of the bot. Alias: p
uptime: Shows the uptime of the bot in Days, Hours and Minutes. Alias: up
stats: Shows bot stats.
avatar: Shows either your's or a user that you mention's avatar. Alias: av
invite: Invite the bot. Alias: inv
spotify: Shows a user's spotify status.
weather:
Shows the weather for a specified location
level: Shows your level. Alias: rank

Utility Commands

Commands that add utility to your server

calculate: Calculates the equation you input. Use * for multiplication and / for division. Alias: calc
serverinfo: Shows basic server info. Alias: si
userid: Gets the ID of a mentioned user. Aliases: id, uid
userinfo: Shows user info of either you or a mentioned user. Aliases: ui, whois
announce: Announce something to the server.
resetroles: Resets a user's roles. Alias: rr

Fun Commands

Commands for fun

8ball: Ask a question and get an answer! Alias: ball
emojify: Turn your text into emojis! Alias: emoji
image: Shows an image from the search term you provide
cat: Gets a cute picture of a cat from random.cat. Aliases: kitten, kitty
dog: Gets a cute picture of a dog from dog.ceo. Aliases: doggy, doggo
minesweeper: Play a fun game of minesweeper! Alias: ms
flip: Flip a coin! Aliases: coin, coinflip, cf
cowsay: Moooo... Alias: cow
fight: Fight someone! Alias: attack
hug: Hugs someone.
meme: Get a meme straight from reddit
chat: Chat with an AI. Restricted to Level 5
